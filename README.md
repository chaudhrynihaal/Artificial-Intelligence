# 🤖 AI Algorithm Implementations

A collection of foundational AI algorithms implemented from scratch in Jupyter Notebooks. Each notebook demonstrates a key concept in artificial intelligence, search strategies, machine learning, or optimization.

These implementations are clean, well-commented, and designed for **learning, teaching, or as a base for client projects** needing custom AI logic.

---

## 📁 Repository Contents

| Notebook | Algorithm / Topic | Use Case |
|----------|------------------|----------|
| `A-STAR.ipynb` | A* Search Algorithm | Pathfinding, graph traversal with heuristics |
| `UNIFORM_COST.ipynb` | Uniform Cost Search | Optimal pathfinding (Dijkstra's algorithm) |
| `MINI-MAX_ALGORITHM.ipynb` | Minimax with optional pruning | Two-player games (e.g., Tic-Tac-Toe, Chess) |
| `GENETIC_ALGO_5-QUEEN-PROBLEM.ipynb` | Genetic Algorithm | Optimization, 5-Queen problem solution |
| `K_MEANS-K_MEDIODS.ipynb` | K-Means & K-Medoids Clustering | Unsupervised learning, customer segmentation |
| `COMPLETE-PERCEPTRON-LEARNING.ipynb` | Perceptron Learning Algorithm | Binary classification, foundational neural network |
| `EVALUATION_MEASURES.ipynb` | Model Evaluation Metrics | Precision, recall, F1, accuracy, confusion matrix |
